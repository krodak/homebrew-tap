# homebrew-tap

Homebrew tap for personal CLI tools.

## Formulae

| Formula | Description |
|---------|-------------|
| `clickup-cli` | ClickUp CLI for task management |
| `sad-cli` | Search Apple Docs - Apple Developer Documentation CLI |
| `sad-cli-bin` | Same as above, pre-built universal binary |

## Install

```bash
brew tap krodak/tap
brew install clickup-cli
brew install sad-cli        # build from source
brew install sad-cli-bin    # pre-built binary (faster)
```

## Upgrade

```bash
brew update
brew upgrade clickup-cli
brew upgrade sad-cli
```
